# Styleguide

## HTML

- DOCTYPE html, lang="ru"
- Отступы — 2 пробела
- Двойные кавычки для атрибутов
- Самостоятельное закрытие тегов без /
- CSS в head, JS — в конце body

## CSS

- Селекторы классов через дефис
- Пустая строка между правилами
- Цвета в hex или названиях
- Вложенность через пробел

## JavaScript

- const/let, без var
- camelCase для переменных и функций
- PascalCase для констант-перечислений
- Точка с запятой обязательна
- Фигурная скобка на той же строке
- Пробелы вокруг операторов, после запятых, внутри объектов
- Однострочные guard-return
- Магические числа выносить в константы
