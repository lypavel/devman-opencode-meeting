# Styleguide

## Общие правила

- Отступы — 2 пробела
- Конечная пустая строка в файле
- Максимальная длина строки — 100 символов
- Все надписи на интерфейсе — на русском языке
- Код и комментарии (если есть) — на английском

## HTML

- DOCTYPE html, lang="ru"
- Отступы — 2 пробела
- Двойные кавычки для атрибутов
- Самостоятельное закрытие тегов (meta, link) без /
- Подключение CSS в head, JS — в конце body

## CSS

- Селекторы классов через дефис: `.win-message`, `.card-inner`
- Одно правило на селектор в multi-line (не группировать)
- Пустая строка между правилами
- Цвета в hex или названиях (не rgb/hsl)
- Вложенность через пробел, без препроцессоров

## JavaScript

- const/let, без var
- camelCase для переменных и функций
- PascalCase только для констант-перечислений (EMOJIS)
- Точка с запятой обязательна
- Фигурная скобка открывается на той же строке
- Пробелы вокруг операторов, после запятых, внутри объектов
- Однострочные guard-return (без скобок)
- Предпочитать чистые функции где возможно
- Избегать магических чисел (выносить в константы)
