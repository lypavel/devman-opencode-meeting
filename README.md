# Memory Game

Игра «Найди пару» (Memory). Поле 4×4, 8 пар эмодзи. Чистый HTML + CSS + JS, без сборщиков и бэкенда.

## Запуск

```bash
./run.sh           # порт 8080
./run.sh 3000      # произвольный порт
```

Открыть `http://localhost:8080/src`.

## Разработка

Ведётся через [opencode](https://opencode.ai). Подробности в [CONTRIBUTING.md](CONTRIBUTING.md).
