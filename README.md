# Memory Game

Игра «Найди пару» (Memory). Поле 4×4, 8 пар эмодзи.

## Запуск

```bash
./run.sh              # порт 8080
./run.sh 3000         # произвольный порт
```

Открыть в браузере `http://localhost:8080`.

## Стек

Чистый HTML + CSS + JS, без сборщиков и бэкенда.

## Разработка

Проект ведётся через ИИ-агента [opencode](https://opencode.ai).
